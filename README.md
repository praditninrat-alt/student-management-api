# Student Management API (.NET 6)

## Overview
This project is a sample ASP.NET Core Web API demonstrating
clean architecture, repository pattern, and scalable backend design.

## Tech Stack
- ASP.NET Core 6
- Entity Framework Core
- SQL Server
- RESTful API

## Architecture
Controller → Service → Repository → Database

## Project Structure
src/
 ├── Controllers
 ├── Services
 ├── Repositories
 ├── Models
 └── DTOs

## Purpose
This repository focuses on backend architecture design and structure.
The implementation can be extended easily for real-world use cases.
